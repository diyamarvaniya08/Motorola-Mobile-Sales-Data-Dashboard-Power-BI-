# Motorola-Mobile-Sales-Data-Dashboard-Power-BI
Interactive Power BI dashboard analyzing Motorola mobile sales data to uncover sales trends, customer behavior, and product performance insights.

# Project Overview
 This project involves building an interactive Power BI dashboard to analyze Motorola mobile sales data and uncover insights related to sales performance, customer behavior, and product trends.
 The project was completed as part of a Power BI Micro Course, applying practical concepts to a real-world sales dataset.
 
# Tools & Technologies
 - Microsoft Power BI
 - Data Cleaning & Data Modeling
 - DAX (Basic Measures)
 - Data Visualization

# Dataset
 - The dataset contains information related to:
     - Mobile models
     - Sales amount and quantity
     - City-wise transactions
     - Payment methods
     - Customer age groups
     - Product ratings
- File: Day - 30 - Mobile Sales Data.xlsx

# Dashboard Features & Insights
 - Total sales, quantity, and number of transactions
 - Month-wise sales trend analysis
 - City-wise sales performance
 - Payment method distribution
 - Top-selling Motorola mobile models
 - Customer ratings and age group analysis
 - Interactive slicers for month, Mobile model, and Brand

# Objective
 - To transform raw sales data into clear, actionable business insights using interactive Power BI dashboards.
